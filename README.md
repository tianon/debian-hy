OK, so, why?
------------

Well. Python is awesome. So awesome, that we have so many tools to alter the
language in a *core* way, but we never use them.

Why?

Well, I wrote Hy to help people realize one thing about Python:

It's really awesome.

Oh, and lisps are neat.
